## Description
This application gets total number of unique pincodes and districts in north eastern states of India.

Data is fetched from <a href="https://www.data.gov.in/" target="_blank">Open Government Data (OGD) Platform India</a>, an official Indian Govt. platform.

### How to use
It's an easy to use application providing better user experience with good UI.
Using this application is just a 2 step process to get desired results.

1. Go to https://muhafiz5814.github.io/distcode/ 
2. Select any one state from the list.

And done! In just two steps user will get total number of unique `pincodes`and `districts` for the selected state.

### Features
1. Easy to use with good UI.
2. Proper error handeling has been implemented for better user experience as well as developer experience.
3. Showing loading message feature has been implemented for better user experience.
